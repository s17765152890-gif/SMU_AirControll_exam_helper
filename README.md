# 大气污染控制工程刷题轻应用

这是一个为《大气污染控制工程》课程复习量身定制的单文件轻量级刷题应用。项目开源免费，只需一个 HTML 文件即可在现代浏览器中直接运行，且原生支持通过 GitHub Pages 一键部署上线。

## 🌟 核心特色

- **开箱即用**：纯静态 HTML/CSS/JS 实现，无任何框架依赖和后端配置，下载即用。
- **现代化 UI**：界面清爽干净，内置响应式设计，完美适配手机端和电脑端屏幕，且支持亮色/暗色主题（Dark Mode）无缝切换。
- **功能完备**：涵盖题目展示、解析查看、选项交互等刷题核心功能。
- **GitHub Pages 友好**：原生结构即可完美支持 GitHub Pages，只需简单设置即可拥有专属的在线刷题链接。

## 🚀 部署指南 (GitHub Pages)

如果你已经将该仓库推送到你自己的 GitHub 账号下，可以按照以下步骤生成在线网页：

1. 在你的 GitHub 仓库页面，点击顶部导航栏的 **Settings**（齿轮图标）。
2. 在左侧菜单中找到并点击 **Pages**。
3. 在 **Build and deployment** 区域，将 **Source** 设置为 **Deploy from a branch**。
4. 将 **Branch** 选择为 `main`，目录选择 `/ (root)`，点击 **Save** 保存。
5. 稍等 1~2 分钟，刷新页面即可在上方看到类似于 `https://<你的用户名>.github.io/SMU_AirControll_exam_helper/` 的在线链接，点击即可访问！

## 🛠️ 本地使用

如果你只想离线使用：
1. 下载或者 Clone 本仓库到本地。
2. 双击通过现代浏览器（Chrome, Edge, Safari 等）打开根目录下的 `index.html` 即可开始刷题。
